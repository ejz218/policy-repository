# Drafts Under Community Review

This directory contains **proposed policy changes** that are currently open for community review.

> [!NOTE]
> To comment on a draft, please use the associated **GitHub Pull Request** — not this file directly. Links to open PRs are listed below and on the [repository home page](../README.md).

---

## How to Review a Draft

1. Find the relevant Pull Request in the [open PRs list](../../pulls?q=is%3Aopen+label%3Apolicy-review).
2. Click **"Files changed"** to see exactly what is being proposed.
3. Leave comments on specific lines or use the general comment box.
4. A free [GitHub account](https://github.com/signup) is required to comment.
