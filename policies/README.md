# Current Policies

This directory contains all **approved, in-effect** policies. Each file represents the current authoritative version of a policy.

Previous versions of any policy are accessible via Git history:
- On GitHub: click **"History"** on any file
- Via CLI: `git log --oneline -- policies/<filename>.md`
